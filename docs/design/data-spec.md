### 5.1 2025 파일·표본 명세 (확정)
| 항목 | 값 | 근거 |
|------|----|----|
| 파일 | `2025_언론수용자조사_데이터.sav` (+동일 내용 .xlsx) | data/raw/audience/2025 |
| 행(N) | **6,000명** | `.sav` shape |
| 변수 수 | **223개** | `.sav` shape |
| 파일 인코딩 | **EUC-KR** (pyreadstat `encoding='euc-kr'` 또는 자동) | meta.file_encoding |

| **가중치 변수** | **`WT`** (레이블 "가중치") | `.sav` ★재검증 대상 해소 |
| 결측·특수코드 | **9997=기타, 9998=해당없음/이용안함, 9999=모름·무응답** | 값레이블 전수 |
| 비공개 처리 | 매체명·언론사명·신문제호 등 자유응답(문5·30·89·90 등)은 .sav 미수록 | 코드북 [비공개] |

### 5.2 인구통계·배경 변수 (확정)
| 변수 | 내용 | 코드/형식 |
|------|------|----------|
| `ID` | 응답자 고유번호 | 정수 |
| `SQ1` | 거주 시도 | 1~17 (1=서울…17=제주) |
| `DQ2` | 성별 | 1=남성, 2=여성 |
| `DQ3` | **연령 = 만 나이 (연속값)** | 정수(예 34) → **분석 시 연령대 비닝 직접 수행** |
| `BQ3` | 최종학력 | 1=중졸↓ 2=고졸 3=대재/대졸 4=대학원↑ |
| `BQ4` | 직업 | 1~12 + 9997(기타) (코드북 직업분류 12종) |
| `BQ5` | 가구소득 | 1(100만↓)~9(800만↑) 9구간 |
| `BQ6` | 주관적 계층 | 1=하~5=상 |
| `BQ7` | 정치성향 | 1=매우진보~5=매우보수 |
| `BQ8` | 정치·사회 관심도 | 1~5 |
| `BQ9` | 제21대 대선 투표여부 | 1=투표 2=미투표 |
| `BQ10` | 정기구독 신문 유무 | 1=있음 2=없음 9997=기타 |

### 5.3 건강지수 4차원 ↔ 변수 가용성 (★ Task 3 핵심 결론, .sav 검증 완료)

| 차원 | .sav 변수 | 가용성 | 비고 |
|------|----------|:----:|------|
| **다양성** | `Q84`(주 이용경로 17종+9998=이용안함) + 매체 **이용여부 게이트 21문항**(Q1·Q6·Q8·Q10·Q13·Q17·Q20·Q36·Q39·Q43·Q46·Q50·Q53·Q58·Q61·Q64·Q67·Q70·Q73·Q76·Q78) | ★★ 가용 | 이용매체 수 카운트·집중도(HHI) 산출 가능. **단, 복수응답 경로 Q80~Q83은 .sav 미수록**(Q84 단일경로만 존재) |
| **신뢰** | `Q85_1~5`(언론 인식) `Q86_1~7`(역할) `Q87_1~2`(전반/실제 신뢰) `Q88_1~3`(출처별 신뢰) `Q91_1~8`(보도 문제점) `Q92_1~5`(매체 언론기능) `Q94_1~4`(언론인) `Q95_1~10`(직업군) `Q96`(사회신뢰) | ★★★ 매우 풍부 | 4차원 중 **최강**. 모두 5점 척도. 요인분석으로 하위차원 정제 가능 |
| **검증습관** | **직접 문항 없음.** 프록시: `Q93`(출처 인지 5점) + `Q34_3`·`Q56_2`(특정 언론사 능동 구독/검색) + `Q34_5`·`Q56_4`(검색 이용) + `Q91_*`(문제점 인식=비판적 리터러시) | △ 프록시만 | ⚠️ **"팩트체크·교차확인" 직접 문항 부재**(.sav "검증/팩트/확인" 검색 0건). 차원 재정의 필요 |
| **회피(역지표)** | **직접 문항 없음.** 프록시: `Q84==9998`(주 경로로 뉴스 이용 안함) + 매체 이용여부 게이트 전반 '안함' | △ 약한 프록시 | ⚠️ **"의도적 뉴스 회피" 직접 문항 부재**(.sav "회피" 검색 0건). 시장조사 '72% 회피'(2024)는 본조사 정규문항 아님 → **회피 차원은 행동지표 프록시로 재설계** |

> **🎯 설계 함의(Task 3 직결)**: 4차원 중 **신뢰·다양성은 견고**, **검증습관·회피는 직접 문항 부재**.
> → 지수 재설계 옵션: (a) **검증습관·회피를 프록시로 조작적 정의**(위 표) 후 타당도 검증, 또는
> (b) 4차원 → **신뢰·다양성 중심 2~3차원으로 축소**하고 검증/회피는 보조 행동지표로 강등.
> 최종 결정은 `notebooks/01-eda-2025.ipynb`에서 프록시 변수 분포·결측·상관 확인 후 확정.
