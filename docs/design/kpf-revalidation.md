## 2. 2019→2020 계단 진단

### 2.1 배제된 가설 (원자료·코드북 검증)

| 가설 | 검증 | 결과 |
|------|------|:----:|
| 문항 워딩 변화 | 문77(2019)·문78(2020) 라벨 원문 대조 | 동일("우리나라 언론은 공정하다" 등 5문항) → **배제** |
| 척도/특수코드 인공물 | 2020 값 라벨의 2.5·5.5·9(해당없음) 코드 실데이터 빈도 | **실데이터에 0건**(1~5만 존재) → **배제** |
| 조사 모드 변화 | 2019 표(CAPI·태블릿PC) ↔ 2020 표 3-1(TAPI·태블릿PC, "TAPI는 CAPI에 속한다") | 동일(가구방문 태블릿 대면면접) → **배제** |
| 조사 시기 차이 | 2019: 6/13~7/18 ↔ 2020: 6/9~7/12 | 동일 계절(6~7월) → **배제** |
| 가중치 산출 차이 | 2019 wt1↔wt2↔비가중 민감도 | 최대차 0.011(공정 2.807/2.818) → **배제** |
| 특정 응답범주 쏠림 | 1~5 분포 이동 형태 | **분포 전반의 평행 이동**(공정: '별로 그렇지 않다' 33.7→24.1, '약간 그렇다' 21.4→31.2) — 코딩 오류형 국소 이동 아님 |

### 2.2 남는 설명 (관측적으로 분해 불가)

1. **코로나19 국면 효과**(기간 효과): 2020.6 조사는 팬데믹 초기 — 재난 정보 수요로 언론 이용·평가가 전반 상승. notebooks/24의 APC 분해("상승=기간 효과", 정렬추세와 r=+0.99)와 정합.
2. **실사대행사 변경**: 2019 (주)한국갤럽 → 2020 (주)칸타코리아. 표집설계·조사도구는 동일하나 실사 운영 차이가 응답 스타일에 영향 가능.
3. 2020 보고서 스스로 신중 해석(p.28): 언론인 평가 등 "전반적으로 상승한 것을 고려하면 … **금년도 자료의 특성일 수 있음**을 염두에 두어야".

### 2.3 판정

- 계단은 **KPF 공식 통계에 실재**하는 조사연도 간 변화 — 하모나이즈·정렬법 파이프라인의 인공물이 아니다.
- 따라서 "2019 대비 2020 언론 인식 상승"은 **검증된 결과**로 인용 가능하되, **원인 귀속(코로나 vs 실사기관) 분해 불가 각주를 의무**로 붙인다.
- 정렬 잠재추세의 **2019 기준연도 의존성**(추세 절반이 첫 스텝) 해석 주의는 기존대로 유지(notebooks/23 DB③).
