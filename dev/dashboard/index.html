MGCFA 교차검증 — 직접구현(numpy/scipy) ↔ semopy
pip 손상기 numpy/scipy로 직접 구현했던 ML CFA 추정기를 복구된 pip로 설치한 독립 라이브러리 semopy 2.3.11로 외부 교차검증. 동일 연도 데이터로 재적합 → 표준적재 직접↔semopy 최대 절대차 0.0003(3지표)·0.0001(4지표), CFI/RMSEA도 소수 4자리까지 일치. 연도간 적재 SD 0.031~0.047(metric 동등 독립근거) + pingouin Cronbach α 0.70~0.82(단일차원 확인). 직접구현 metric 동등 강지지가 독립 검증됨. 한계: 정렬법(alignment) 교차검증은 R sirt/Mplus 영역으로 범위 외 명시.
