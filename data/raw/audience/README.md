# data/raw/audience/ — 언론수용자 조사 (연도별)

대회 **메인 데이터**. 연도별 하위 폴더(`2019/` ~ `2025/`)에 신(新) 체계 7개년을 보관합니다.

## 파일명 규칙
```
<연도>/<연도>_언론수용자조사_<종류>.<확장자>
종류: 데이터(.sav/.csv/.xlsx) · 설문지겸코드북(.pdf) · 통계표(.pdf) · 보고서(.pdf)
```
- `.sav`(SPSS) 로딩 권장 — 값/변수 레이블 보존(`pyreadstat`).
- **2022는 개인용/가구용 분리** → `개인용`/`가구용` 접미사. 분석 단위=개인이면 **개인용** 사용.
- 연도별 포맷 차이: 2023·2025는 `.xlsx`, 그 외 `.csv` 제공.
- PDF(코드북/통계표/보고서)는 git 미포함(대용량). 변수 해석엔 코드북 PDF가 필수 → 로컬에서 별도 확보.
