# KPF 원시자료 다운로드 가이드 (수동)

> 다운로드 사이트: https://www.kpf.or.kr/front/mediaStats/mediaStatsListPage.do
> ⚠️ KPF는 다운로드 전 **연구활용 동의 절차**를 요구하므로 **자동 수집 불가 → 본인이 직접 받아야 합니다.**
> 받은 파일은 이 폴더(`data/raw/`)에 아래 권장 파일명으로 저장하세요. (대용량/저작권 → git 커밋 금지, `.gitignore` 대상)

## 다운로드 방법
1. 위 목록 페이지 접속 → 상단 필터에서 **구분: 수용자조사**, **자료구분: 원본 데이터** 선택(검색 범위 축소).
2. 아래 표의 **정확한 제목**으로 항목을 찾아 클릭 → 연구활용 동의 → ZIP 다운로드.
3. 압축 해제 후 `data/raw/<권장 폴더>`에 저장.
