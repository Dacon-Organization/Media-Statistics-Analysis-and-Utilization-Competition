# web/ — "뉴스 건강검진" 웹 데모

Next.js 15 기반 실동작 데모. **진단 플로우 1개에 집중**(대중투표·실현가능성 증명용).

## 원칙
- **DB·백엔드 없이** 분석에서 추출한 계수/기준값을 **브라우저에서 계산** → Vercel 정적 배포로 안정성 확보.
- 히어로 기능(개인 뉴스 건강점수→페르소나→맞춤 처방)만 구현. 계정·연도추적·관리자 대시보드는 설계 문서로 비전 제시.
- 스택: Next.js 15(App Router) + TypeScript(strict) + Tailwind + shadcn/ui.

## 실행(예정)
```
npm install && npm run dev      # 개발
npm run build                   # 빌드
```
> 아직 스캐폴딩 전. 분석에서 지수 계수가 확정된 뒤 착수.
